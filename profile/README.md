## Fixing Common Setup or Scan Issues

- Scan connection failing? Confirm credentials, network access, and repository permissions before starting discovery again.  
- Missing results? Review source selection, file scope, and data classification rules to make sure the intended systems are included.  
- Reports look incomplete? Re-run sensitive data discovery after policy updates and verify that data governance filters are not excluding key sources.  
- Slow analysis? Limit the initial scope, close unused applications, and review larger enterprise data protection sets in smaller groups.  
- Access errors? Check administrative roles, backup security permissions, and supported integration settings for the connected environment.
